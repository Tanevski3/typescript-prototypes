> Basic Typescript Webpack Prototype

## Install

```sh
$ npm install
```

## Build Development

```sh
$ npm run build-dev
```

or

```sh
$ npm run watch-dev
```

## Usage

```sh
$ npm run serve-dev
```

**Open http://localhost:8080 in browser and click on dist/ directory**
